Installation

SpaFEDTe should compile and work at least on Linux and Mac OSX.

SpaFEDTe Requirements

In order to successfully use SpaFEDTe, the following software has to be installed in your system, each with it's own dependencies:

  • MPICH2
  • PETSc (download version 3.2)
    • f2cblaslapack
  • MOAB (download version 4.1)
    • HDF5 (default MOAB parallel mesh format)
      • zlib
      • SZIP
    • Zoltan (for MOAB mesh partitioning capabilities)
      • parMETIS
  • Eigen3

Compiling from source

Some details on how to compile SpaFEDTe with dependencies can be found in the README.INSTALL file in the main directory.
